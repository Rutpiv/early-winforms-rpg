# Windows Forms RPG

Early academic RPG built in C# with Windows Forms.

## Overview

This project was developed individually during the second semester of college,
as part of the Algorithms II course. At the time, it was created before formal
exposure to object-oriented programming, based on self-study of Microsoft C#
documentation using Visual Studio 2019.

## Context

- Individual academic project
- Algorithms II (second semester of college)
- C# with Windows Forms
- Visual Studio 2019
- Focus on algorithms and game logic

## Notes

This repository is preserved as a snapshot of an early learning stage and is kept
intentionally close to its original structure.
